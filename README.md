# MLH2_Hydrides
This work is linked to the application of machine learning in hydrigen absorption chemical solutions. We study the absorption on metal hydrides using several ML techniques basically built on the tree model.
# Data Features
Composition Formula
Hydrogen Weight Percent	
Heat of Formation (kJ/mol H2)
Temperature (oC)
Pressure (Atmospheres Absolute)
